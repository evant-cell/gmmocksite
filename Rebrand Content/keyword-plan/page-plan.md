# GM Per-Page Keyword Plan

One entry per page in the rebrand architecture. Data: Ahrefs US, pulled 2026-06-27. Format per page: **Job**, **Primary** (volume/KD), **Secondary**, **AEO questions** (for the definition block and FAQ), **Links** (out and in), **Priority**, **Schema**.

Priority key: **P1** = quick win, write first (low KD and high commercial or AEO value). **P2** = core target, needs cluster support. **P3** = brand or thin-volume, lower SEO urgency.

Volume note: the "40%/5x/$1/ten times" style numbers in copy must be GM's real figures, not invented. Keyword volumes below are search demand, not GM stats.

GSC overlay: `gsc-insights.md` has GM's real current rankings, the striking-distance quick wins (impression tracking, mer vs roas, programmatic cpm trends already pos 6 to 10), and the existing-URL redirect map. Several pages here already exist and rank (/how-it-works/, /the-dashboard/, /pricing/, /faqs/, /our-platform/, the behavior-targeting page). Check the redirect map before changing any URL.
